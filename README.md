Este proyecto consiste en un programa de Python que permite analizar información de videos almacenados en un archivo CVS . Esta diseñado para facilitar la exploracion de estos datos mediante un menu interactivo que ofrece diferentes funciones de consulta y filtrado. Despues de cargar el archivo, se leen asta 50 resgistros, de los cuales se pueden realizar busquedas especificas (sin importar mayusculas o minusculas ) y conocer el numero de concidencias. Ademas, el programa puede calcular estadisticas como el video con mayor número de vistas, el de menor número de vistas, el promedio de visualizaciones, la cantidad de registros procesados, contar cuantos videos hay de un idioma especifico, enlistar los idiomas encontrados segun su frecuencia o filtrar los videos segun un minimo de vistas definido por el usuario.Para su ejeución, es necesario contar con Python 3(sin librerias externas),el sistema se ejecuta desde la consola y presenta un menú con seis opciones principales: búsqueda, estadísticas, filtrado por vistas, análisis de idioma específico, resumen de idiomas y salida del programa.
